Quick Start

Quick Local Install

RAPIDS offers several installation methods, the quickest is shown below. For more detailed guides go to our install page

Requirements

A. NVIDIA Pascalâ„¢ or better GPU with a compute capability 6.0 and above

B. Ubuntu 20.04, CentOS, Rocky Linux 8 or via WSL2 on Windows 11

C. Recent CUDA-version and NVIDIA-driver pairs. Check yours with: nvidia-smi

Install with Conda

1. Download and run the conda install script: 

wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh

2. Then quick install rapids with:

conda create -n rapids-22.08 -c rapidsai -c nvidia -c conda-forge \ rapids=22.08 python=3.9 cudatoolkit=11.5

Install with PIP

1. Install via pip channels:

pip install cudf-cu11 dask-cudf-cu11 --extra-index-url=https://pypi.ngc.nvidia.com
pip install cuml-cu11 --extra-index-url=https://pypi.ngc.nvidia.com
pip install cugraph-cu11 --extra-index-url=https://pypi.ngc.nvidia.com
