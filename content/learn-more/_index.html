About

RAPIDS is a collection of open source software libraries and APIs that gives you the ability to execute end-to-end data science and analytics pipelines entirely on NVIDIA GPUs.

About The Project

Familiar

Utilizing NVIDIA CUDA primitives for low-level compute optimization, RAPIDS exposes GPU parallelism and high-bandwidth memory speed through user-friendly interfaces:

  • Dataframe processing with Pandas like cuDF
  • Machine learning with SciKit-Learn like cuML
  • Graph processing with NetworkX like cuGraph/li>
  • Spatial analytics with GeoPands like cuSpatial
  • Image processing with scikit-image like cuCIM
  • Low level compute primitives with RAFT
  • Signal processing with SciPy like cuSignal
  • Seamless cross-filtered dashboards with cuxfilter
  • Apache Spark acceleration with Spark RAPIDS

and more...

Scalable

RAPIDS delivers impressive acceleration from on a single GPU desktop system all the way to MNMG (multi-node multi-GPU) configurations on an expansive range of NVIDIA hardware.

  • single GPU with cuDF
  • Multi-GPU with Dask-cuDF
  • Large clusters with RAPIDS-Spark
  • HPC with RAPIDS and Slurm
and more...

Open Source

RAPIDS had its start from the Apache Arrow and GoAi projects based on a columnar, in-memory data structure that delivers efficient and fast data interchange with flexibility to support complex data models. With NVIDIA backing, it has maintained a focus on open source development (typically Apache 2.0 licensed) and collaboration with a broad community of cutting edge data science projects.
